# 🌐 Maine-Kho — Website Trải Nghiệm Tương Tác & Giao Diện Sáng Tạo

Maine-Kho là một dự án website frontend được xây dựng bằng HTML, CSS và JavaScript, tập trung vào các hiệu ứng tương tác, giao diện sáng tạo và trải nghiệm người dùng mang tính giải trí. Đây là một project mang tính thực hành, giúp rèn luyện kỹ năng thiết kế web và xử lý animation cơ bản.

## ✨ Điểm Nổi Bật

* 🎨 Giao diện web nhiều trang (multi-page UI)
* ✨ Hiệu ứng tương tác và animation thú vị
* 🎵 Tích hợp âm thanh và trải nghiệm multimedia
* 💌 Các trang chức năng như mail, hiệu ứng love, mắt (mat),…
* 📱 Thiết kế đơn giản nhưng sáng tạo
* ⚡ Chạy trực tiếp trên trình duyệt, không cần backend

## 🏗️ Cấu Trúc Dự Án

Dự án được tổ chức theo dạng website tĩnh:

### Giao diện chính

* `index.html`, `index1.html`, `index2.html`
* Điều hướng giữa nhiều trang trải nghiệm

### Thành phần chức năng

* `mail.html` — giao diện gửi/hiển thị nội dung mail
* `love/` — hiệu ứng tình cảm, tương tác
* `mat/` — hiệu ứng hình ảnh/animation
* `sound/` — âm thanh và hiệu ứng audio

### Tài nguyên

* `css/` — style chính và phụ
* `js/`, `script.js`, `app.js` — logic tương tác
* `image/`, `image1/`, `img/` — tài nguyên hình ảnh

## 💻 Công Nghệ Sử Dụng

* HTML5
* CSS3
* JavaScript (Vanilla)
* Media Assets (audio & images)
* Static Web Architecture

## 🎯 Mục Tiêu Dự Án

Maine-Kho được xây dựng nhằm:

* Thực hành kỹ năng frontend cơ bản
* Hiểu cách tạo website nhiều trang
* Làm quen với animation và DOM manipulation
* Xây dựng trải nghiệm web sáng tạo, giải trí

## 🚀 Định Hướng Phát Triển

* Nâng cấp UI theo phong cách hiện đại
* Tối ưu responsive cho mobile
* Chuyển sang React/Vue architecture
* Thêm backend lưu dữ liệu tương tác
* Tích hợp hiệu ứng 3D hoặc WebGL
* Cải thiện UX và tốc độ tải trang
